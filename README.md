# 모의고사 정시 합격률 대시보드

YMABI/SearchUnivMajorPossibility의 공개 엑셀 데이터를 웹에서 쉽게 볼 수 있도록 만든 MVP입니다.

## 주요 기능

- 국어, 수학 선택과목, 영어, 한국사, 탐구 2과목, 제2외국어/한문 성적 입력
- 2024~2026학년도 정시 비교 기준 선택
- 대학권역, 대학명, 학과명, 최소 합격률 필터
- 학과별 예상 합격률, 컷 대비 차이, 모집인원, 경쟁률 표시
- 90% 이상 안정권 후보만 따로 보기

## 데이터 갱신

1. `.source-data` 폴더에 최신 `27학년도 정시 지원가능 대학 및 학과 검색.xlsb` 파일을 둡니다.
2. 아래 명령으로 웹앱용 JSON을 다시 만듭니다.

```bash
python scripts/extract_data.py
```

원본 파일이 로컬에 없으면 스크립트가 YMABI GitHub 원본에서 자동으로 내려받습니다.

## 자동 업데이트

GitHub Actions가 매일 04:00 KST에 원본 데이터를 다시 받아 무료 GitHub Pages에 재배포합니다. 별도 Secret은 필요하지 않습니다.

자세한 설정은 `docs/auto-update.md`를 참고하세요.

## 계산 반영 범위

현재 웹앱은 원본 VBA 계산 순서와 DB를 웹 계산 엔진으로 옮겼습니다.

- 시험 월별 표준점수 등급 구간
- 대학별 백분위/표준점수/변환표준점수 기준
- 2024~2026 정시 변환표준점수표
- 국어/수학/영어/한국사/탐구 반영비율
- 27개 반영비율 열 전체
- 영어/한국사/제2외국어 등급 환산표
- 수학 선택과목 가산점
- 14개 가산점 열 전체
- 응시과목 지정 조건
- 62개 대학별 전용 환산 절차
- 엑셀의 50%·70% 컷 기반 합격가능성 판정 문구

원본 `.xlsb`의 VBA는 브라우저에서 직접 실행하지 않습니다. 계산에 필요한 안전한 로직만 JavaScript로 번역하고, 매일 최신 DB와 함께 재검증합니다.

## 참고

원본 엑셀 자체도 수정될 수 있으므로 실제 지원 전에는 대학 공식 입시자료를 함께 확인해야 합니다.

## 원본 검증 상태

웹 화면에는 `data/audit.json`과 `data/integrity.json` 기준의 검증 현황을 표시합니다.

- 2026년 7월 13일 최신 원본은 성균관대학교 결과 30개를 저장하고 있습니다.
- 30개 모두 환산점수 0.005점 이내와 합격가능성 문구까지 완전히 일치합니다.
- 전체 8,516개 모집단위는 계산 오류나 빈 결과가 없는지 자동 검사합니다.
- 원본에 저장된 독립 비교값이 없는 대학은 화면에서 `검증 대기`로 구분합니다.
- 전체 대학의 독립 Excel 실행 기준값이 확보되기 전까지 `100% 검증 완료`라고 표시하지 않습니다.

자동 업데이트는 매일 새벽 최신 `.xlsb`를 다시 받아 `admissions.json`, `audit.json`, `integrity.json`을 함께 재생성하도록 구성되어 있습니다.
