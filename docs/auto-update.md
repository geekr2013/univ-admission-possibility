# 자동 업데이트 설정

이 앱은 GitHub Actions로 하루 1회 원본 데이터를 다시 받아 무료 GitHub Pages에 배포합니다.

## 실행 시간

- 매일 04:00 KST
- GitHub Actions의 cron은 UTC 기준이라 워크플로에는 `0 19 * * *`로 설정되어 있습니다.
- GitHub 화면에서 `workflow_dispatch`로 수동 실행도 가능합니다.

## 노트북 전원

GitHub Actions와 Vercel에서 실행되므로 노트북이 꺼져 있어도 작동합니다.

별도 유료 서비스나 Secret 등록이 필요하지 않습니다.

## 배치 결과물

매일 배치는 아래 파일을 함께 갱신합니다.

- `data/admissions.json`: 모집단위, 과목, 시험월, 변환표준점수 데이터
- `data/audit.json`: 원본 엑셀 `대학별검색` 저장값과 VBA 번역 엔진의 정밀 대조 결과
- `data/integrity.json`: 원본 파일 해시, 시트 범위, VBA 포함 여부, 검증 요약

원본 파일에 7월 등 새 모의고사 성적 구간이 추가되면 `모의고사학생입력성적유효범위` 시트에서 자동 추출되어 시험 정보 선택창에 반영됩니다.

원본이 바뀔 때마다 27개 반영비율, 14개 가산점, 수능 등급표, 변환표준점수표도 함께 다시 추출되고 검증 결과가 갱신됩니다.
